Gráfico de horizonte (horizon graph)

Descripción

Un gráfico de horizonte es una visualización de datos bidimensional que muestra datos cuantitativos durante un intervalo continuo, generalmente un período de tiempo.

Es útil para identificar tendencias y valores extremos en grandes. Su ventaja reside en proporcionar una visión general y destacar patrones y valores atípicos en los datos.

Origen

Desarrollado por Takafumi Saito de Universidad de Agricultura y Tecnología de Tokio.

Se comercializó Panopticon Software quien le dio el nombre.

Funcionamiento

  • Se lee como un mapa de calor a lo largo del tiempo.
  • Se usa colores divergentes para diferenciar valores positivos y negativos.
  • Los colores más oscuros representan valores absolutos más altos.
  • Se obtiene una capa adicional de información con las líneas de tendencia rellenas.

Ejemplos típicos de aplicación

Su diseño compacto facilita la comparación de diferentes series de datos dentro de una misma visualización.

Los gráficos de horizonte facilitan la reducción del espacio vertical, esto permite consolidar un volumen considerable de datos en una sola visualización.

Tipo y estructura de datos

Tabla en formato largo. 3 columnas:

  • Variable temporal: para eje X
  • Variable categórica: para eje Y
  • Variable numérica: para las áreas.

Limitaciones

No ser la visualización más adecuada para identificar con precisión valores específicos.
