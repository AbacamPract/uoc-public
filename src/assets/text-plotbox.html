Diagrama de caja y bigotes (box plot)

Descripción

Resume la distribución de una variable numérica en uno o más grupos. Destaca tendencias centrales e identificar valores atípicos

Origen

Fue creado por el matemático estadounidense John Tukey y presentado en su libro " Exploratory Data Analysis " en 1977.

Funcionamiento

  • La marca central indica la mediana (percentil 50).
  • Los extremos inferior y superior de la caja indican los percentiles 25 y 75, respectivamente.
  • La caja nos indica el rango intercuartil (RIC).
  • Los bigotes se extienden hasta los puntos de datos más extremos que no se consideran valores atípicos.
  • Los valores atípicos se representan individualmente

Ejemplos típicos de aplicación

  • Análisis datos crudos de una variable numérica.
  • Ver cómo se distribuye una misma variable numérica en diferentes valores de una misma variable categórica. Por ejemplo, el salario entre distintos países.

Tipo y estructura de datos

Una o más variables numéricas.

También puede ir con variable categóricas como elemento distintivo entre grupos.

Limitaciones

Puede conllevar la pérdida de información importante. Distribuciones diferentes pueden mostrar el mismo boxplot (Paradoja de Stein). Para resolverlo:

  • Si el conjunto de datos no es muy grande, se podría añadir la nube de puntos.
  • Si el conjunto de datos es grande, se puede superponer un diagrama de violín, que ilustra la distribución.
